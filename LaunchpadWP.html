NOVATION – LAUNCHPAD

CLIP LAUNCHER

The Play/Stop function is mapped to the up arrow (top row, left most button).
The Mode/View function is intended to be pressed to cycle between available sub modes within a primary mode. For now it is only used to change the starting note for the midi note playing area.
Any empty clip and armed track can be recorded with one press of a button in the first four rows.
The buttons marked as "Function" buttons should announce what they do when you press them, via a popup notification. Most of them do nothing so far.
Function B8 changes the Loop Quantize amount between values of 1/4, 1/2, 1, 2, 4.
Press again to go into clip playback. Press any playing clip to stop it.
Overdub feature not yet implemented.

The bottom four rows of the grid are a note input area.
